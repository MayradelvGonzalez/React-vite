Hook 
Es una funcion, que permite controlar el estado de un componente y acceder a su ciclo de vida. 
UseState: 
- Devuelve dos valores, uno para la actualizacion del valor y otro para leer el valor actual.
import React, { useState } from 'react';

function ExampleComponent() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
